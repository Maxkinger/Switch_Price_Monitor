# 公开偏好设置页设计规格

## 1. 文档控制

| 项目 | 内容 |

| 范围 | 已登录管理员的公开偏好设置页面 |

## 2. 目标与非目标

本阶段让管理员在网页内读取和保存已存在的公开设置：启用地区、默认搜索区、主题、时区、日报时间、美国税务州与价格历史保留策略。页面采用单页分组表单和一次保存，避免地区与默认搜索区被拆分为多个请求后产生短暂的无效组合。

本阶段不保存、展示或测试 Telegram Token、Chat ID、恢复码、密码、会话或任何第三方价格站配置。Telegram 相关设置将在生产部署、D1 迁移、Cron 和官方采集流程完成受控验收后另行设计；第三方价格站继续保持未接入状态。

## 3. 页面与交互

### 3.1 导航与加载

- 左侧“设置”入口改为可用导航，地址为 `/settings`；浏览器前进、返回和刷新必须恢复该页面。
- 页面加载时调用同源 `GET /api/settings`，只展示 Worker 返回的 `AppSettings` 公开字段。
- 读取期间显示加载状态；`401` 交由认证外壳清理内存并返回登录；其他错误仅显示 Worker 的安全中文摘要，不展示请求正文、Cookie 或内部错误。

### 3.2 单页分组表单

页面按照以下三组呈现，所有输入均为受控 React 状态：

1. **地区与搜索**：美国、日本、墨西哥、巴西、香港五个复选框，以及默认搜索区下拉框。默认搜索区选项只来自当前勾选地区；最后一个启用地区不可取消，以避免浏览器产生空范围。
2. **展示与日报**：主题下拉框（温暖游戏库、沉稳深色、清爽工具）、IANA 时区文本或下拉输入、`HH:mm` 日报时间与两位美国州代码。主题在本阶段仅保存偏好；实际全局主题样式切换留给独立视觉实现任务，页面不得假装已切换。
3. **数据保留**：永久、最近一年、最近两年的单选选项。保存后只影响后续定时清理，不删除页面加载前已有快照。

页面底部只有一个“保存设置”按钮。修改后统一 `PATCH /api/settings`；保存中禁用提交，成功后使用服务端返回的完整设置重置表单，并显示“设置已保存”。

### 3.3 失败与边界

- 若当前默认搜索区因取消勾选而不再启用，浏览器立即把默认搜索区切换为剩余地区中的第一项；Worker 仍以 `422` 作为最终业务约束。
- `422` 时保留用户未保存的表单内容，并显示 Worker 返回的中文校验摘要，便于修正而不丢失其他分组选择。
- `409 SETUP_REQUIRED` 显示首次设置尚未完成的安全摘要；不会尝试自行写入管理员配置。
- 设置页不包含“保存 Telegram”“发送测试消息”或第三方开关，从 UI、客户端 DTO 和网络请求三个层面防止秘密或未获准来源进入本阶段。

## 4. 数据与组件边界

- 复用既有 `AppSettings`、`RegionCode`、`Theme` 和 `GET/PATCH /api/settings`；不增加 D1 表、迁移、Worker 路由或秘密字段。
- 新建轻量同源设置 API 客户端，只接受固定 `/api/settings` 路径，强制 `credentials: "same-origin"`，并将 Worker 错误转换为仅含状态码与安全文案的前端错误。
- 新建纯设置表单状态模块，负责从 `AppSettings` 初始化、切换地区、保持默认区有效并构造完整公开 PATCH。该模块不保存 Cookie、`createdAt` 或任意秘密。
- 设置页只编排读取、受控编辑和一次保存；导航继续由 `AppShell` 的 History API 统一管理。

## 5. 验收与测试

- 纯状态测试：最后一个地区不能取消；取消当前默认区会选择新的有效默认区；完整 PATCH 不包含 `createdAt`、认证或 Telegram 字段。
- API 客户端测试：GET/PATCH 均为同源、带浏览器凭据；非成功响应只保留 Worker 的安全摘要与状态码。
- 页面测试：加载回填、三组编辑、一次保存、保存中禁用、成功后回填服务器值、`422` 保留草稿、`401` 调用认证失效回调。
- 路由测试：`/settings` 被解析，左侧设置入口可到达，未知路径不显示设置页。
- 浏览器验收：桌面与窄屏下检查分组可读、默认搜索区联动、成功提示与侧栏导航；测试不输入任何真实 Telegram 或管理员秘密。

## 6. 影响范围

- 前端：应用路由、应用外壳、设置 API 客户端、设置表单状态、设置页面、暖色响应式样式与相应测试。
- Worker：复用既有设置接口和服务，无新增路由、迁移或秘密存储。
- 文档：实施完成时更新 API 设计、需求追踪和文档中心状态。
